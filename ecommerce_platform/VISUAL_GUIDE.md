### Binary Search Tree (Products)
```
Insert:  Product(id=1) â†’ BST.insert(1, product)
         Product(id=2) â†’ BST.insert(2, product)
         Product(id=3) â†’ BST.insert(3, product)

        â”Œâ”€â”€â”€ 2 â”€â”€â”€â”
        â”‚          â”‚
       1          3

         BST.search(3) â†’ Returns Product(id=3) in O(log n)

```

### Hash Table (Inventory)
```
Insert:  HashTable.insert(1, 10)  â†’ Product 1: 10 stock
         HashTable.insert(2, 50)  â†’ Product 2: 50 stock

Search:  HashTable.search(1) â†’ Returns 10 in O(1)
         HashTable.search(2) â†’ Returns 50 in O(1)

Update:  HashTable.insert(1, 8)   â†’ Product 1: 8 stock
```

### Queue (Shopping Cart)
```
Initial: []

Enqueue: Queue.enqueue(Mouse)      â†’ [Mouse]
         Queue.enqueue(Keyboard)   â†’ [Mouse, Keyboard]
         Queue.enqueue(Monitor)    â†’ [Mouse, Keyboard, Monitor]

         Result: [Keyboard, Monitor]
```

### LinkedList (Order History)
```

        Order#2 â†’ head â†’ Order#1 â†’ Order#2
        Order#3 â†’ head â†’ Order#1 â†’ Order#2 â†’ Order#3

### Stack (Recent Orders)
```
Push: Order#1 â†’ [Order#1]
      Order#2 â†’ [Order#1, Order#2]
      Order#3 â†’ [Order#1, Order#2, Order#3]

Get Items: Stack.get_items() â†’ [Order#3, Order#2, Order#1] (LIFO)
